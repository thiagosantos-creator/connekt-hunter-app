# Escopo inicial da plataforma Connekt Hunter

## Objetivo

Estabelecer a base técnica e documental do produto operacional da Connekt Hunter para evolução segura por IA e por times humanos.

## Perfis atendidos

- Administrador
- Headhunter
- Cliente / Usuário Empresa
- Candidato

## Escopo desta fase

- estrutura do monorepo
- apps frontend e backend scaffoldados
- pacote de dados com Prisma
- infra local com Docker Compose
- documentação SDD real e modular

## Fora de escopo nesta fase

- site institucional / landing page
- regras finais de negócio por módulo
- experiência visual final
- integrações em produção
