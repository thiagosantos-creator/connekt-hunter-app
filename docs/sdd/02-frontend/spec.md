# Estratégia de frontend

## Backoffice Web

Aplicação React/Next.js para operação autenticada com foco em produtividade, navegação clara e placeholders funcionais para os domínios centrais do produto.

## Candidate Web

Aplicação React/Next.js focada na jornada do candidato, começando por entradas controladas por token e por vaga pública.

## Diretrizes

- usar rotas explícitas para cada área placeholder
- privilegiar componentes compartilháveis em `packages/ui`
- manter o site institucional fora deste repositório
- evoluir autenticação, onboarding e entrevista inteligente a partir das specs por módulo
